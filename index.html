By: Bhavi Kenia

Contact: bckenia@dons.usfca.edu

Project Description

Cybersecurity is a growing global concern, with increasing incidents affecting various industries and causing significant financial losses. This project aims to visualize cybersecurity threats over time and identify patterns within the data.

Project Objectives

  • Analyze trends in cybersecurity threats across different industries.
  • Examine the financial impact of security breaches globally.
  • Identify common vulnerabilities and defense mechanisms.
  • Visualize the global distribution of cybersecurity threats.
